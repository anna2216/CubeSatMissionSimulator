# 🛰 CubeSat Mission Simulator

A web-based CubeSat Mission Simulator developed using HTML, CSS, and JavaScript.

## Features
- 🌍 Orbit Simulation
- 🔋 Battery Monitoring
- 📡 Communication Status
- 📝 Mission Log
- 📷 Image Capture
- 🌡 Data Collection
- 🚀 Data Transmission

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project demonstrates the basic concepts of CubeSat mission operations through an interactive dashboard. It simulates satellite activities such as battery monitoring, mission logging, and communication in an educational environment.
